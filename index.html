Instructions

Server Search

For the server search you need to first type in the server name, then a comma, and then the ip address of the server. Once you have typed it in click on the magnifying glass to confirm your search.
*Note : If any part of the Ip address is mistyped the search will not work

Potential Valid Searches
Bridger.land,bridger.land , Complex,hub.mc-complex.com , Minehut,minehut.com

Player Search

For the player search you have the choice to either type in the players username or their uuid. Once you have typed it in click on the magnifying glass to confirm your search.
*Note : The username and uuid are not case sensitive so don't worry about capitalizing.

Potential Valid Searches
Tubbo_ ,353adc55d4e442edab64cc60f59012a5, LightningWielder
